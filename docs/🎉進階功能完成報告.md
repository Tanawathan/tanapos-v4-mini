## 🔧 技術架構

### 前端技術棧
- **React 18.3.1** - 現代化前端框架
- **TypeScript** - 型別安全
- **Tailwind CSS** - 響應式設計和暗黑模式
- **Vite 7.0.6** - 快速構建工具
- **React Router** - 單頁應用路由

### 後端技術棧
- **Supabase** - 雲端 PostgreSQL 資料庫
- **Realtime Subscriptions** - 即時數據同步
- **Row Level Security** - 資料安全

### 新增資料庫表格
```sql
- payments (付款記錄)
- invoices (發票)
- receipts (收據)
- reservations (預約)
- table_sessions (桌位使用會話)
- table_status_history (桌位狀態歷史)
```

## 🚀 系統功能流程

### 訂單生命週期
```
下單 → 確認 → 製作 → 完成 → 結帳 → 完成
    ↓      ↓      ↓      ↓      ↓
  預約桌位 → 訂單管理 → KDS → 桌位釋放 → 收據列印
```

### 桌位管理流程
```
空閒 → 預約/使用中 → 用餐 → 清潔 → 空閒
    ↓        ↓        ↓      ↓
  預約管理   →  換桌/合桌  →  結帳  →  週轉統計
```

### 結帳系統流程
```
選擇訂單 → 確認金額 → 選擇付款方式 → 處理付款 → 開立發票/收據 → 完成
```
