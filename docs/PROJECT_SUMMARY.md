## 🏗️ 技術架構

```
TanaPOS V4-Mini/
├── 前端框架: React 18.3.1
├── 開發語言: TypeScript
├── 樣式框架: Tailwind CSS
├── 建構工具: Vite
├── 資料庫: Supabase (PostgreSQL)
├── 狀態管理: React Hooks
└── 路由管理: React Router v6
```

## 📁 專案結構

```
src/
├── components/          # React 組件
│   ├── Dashboard.tsx    # 主儀表板
│   ├── Reports.tsx      # 營業報表
│   └── Layout.tsx       # 布局組件
├── config/              # 配置文件
│   └── index.ts         # 系統設定
├── lib/                 # 核心函式庫
│   ├── supabase.ts      # Supabase 客戶端
│   └── store.ts         # 狀態管理
└── App.tsx              # 主應用程式
```

## 🚀 快速啟動

### 1. 啟動開發伺服器
```bash
cd tanapos-v4-mini
npm run dev
```

### 2. 訪問應用程式
- 本地網址: http://localhost:5174
- 網路網址: http://192.168.1.100:5174

### 3. 主要功能導覽
1. **主儀表板** - 首頁，顯示營運概況
2. **營業報表** - 點擊 "營業報表" 卡片
3. **其他功能** - 點擊相應卡片（顯示開發中提示）

## 🎯 核心功能特色

### 1. 智能儀表板
- **即時統計**: 今日訂單、營收、待處理訂單、占用桌台
- **系統狀態**: 自動檢測在線/離線狀態
- **時間感知**: 根據時間顯示不同問候語
- **模擬數據**: 離線時自動切換到演示數據

### 2. 營業報表
- **多時間範圍**: 今日/本週/本月報表
- **關鍵指標**: 總訂單、總營收、平均訂單金額
- **熱銷分析**: 商品銷售排行榜
- **視覺化**: 清晰的數據表格展示

### 3. 響應式設計
- **多螢幕適配**: 桌面、平板、手機全支援
- **現代化UI**: 簡潔直觀的使用者介面
- **快速載入**: 優化的性能表現

## 📱 使用者體驗

### 主儀表板操作
1. **狀態監控**: 左上角顯示系統連線狀態
2. **時間資訊**: 右上角顯示即時時間和日期
3. **數據卡片**: 4個關鍵指標一目了然
4. **功能入口**: 6個主要功能模組快速訪問

### 報表系統操作
1. **時間選擇**: 頂部切換今日/本週/本月
2. **數據總覽**: 3個關鍵營收指標
3. **排行榜**: 熱銷商品詳細列表
4. **返回首頁**: 左上角返回按鈕

## 🔧 設定與配置

### 系統模式切換
編輯 `src/config/index.ts`:
```typescript
export const APP_CONFIG = {
  OFFLINE_MODE: false, // 改為 true 啟用離線模式
  AUTO_REFRESH_INTERVAL: 30000, // 數據刷新間隔
  // ... 其他設定
}
```

### 功能開關
```typescript
FEATURES: {
  DASHBOARD: true,      // 儀表板
  REPORTS: true,        // 報表系統
  POS_SYSTEM: false,    // POS系統（開發中）
  // ... 其他功能
}
```
