## 🚀 系統特色

### 智慧庫存管理
- **雙重庫存追蹤**: 實際庫存 + 虛擬可用量
- **自動成本計算**: 基於配方的動態成本分析
- **多層級生產**: 支援原料→半成品→成品的完整流程

### 彈性配方系統
- **混合模式**: 預製品 + 現場製作
- **批次生產**: 支援大量生產和即時製作
- **成本優化**: 自動計算最佳成本組合

### 即時監控
- **庫存預警**: 低庫存自動提醒
- **生產建議**: 基於需求的智慧生產建議
- **成本追蹤**: 即時成本變化監控

## 📱 使用方式

1. **啟動系統**: 開發伺服器運行在 http://localhost:5174
2. **進入庫存**: 點擊主頁的「庫存管理」或直接訪問 `/inventory`
3. **檢視概覽**: 查看整體庫存狀況和預警
4. **管理原料**: 新增、編輯、調整原料庫存
5. **管理半成品**: 監控半成品狀態和生產需求
6. **配方管理**: 建立和修改生產配方
