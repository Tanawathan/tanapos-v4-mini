# 📖 教學系統功能說明

## 🎯 功能概述

本次更新為 TanaPOS V4-Mini 系統添加了完整的教學功能，包括：

1. **使用說明頁面** - 提供系統基本使用指南
2. **操作指南頁面** - 提供詳細的操作步驟說明
3. **教學模式開關** - 在後台設定中控制教學標籤的顯示

## 🔧 功能實現

### 1. 新增的功能開關

在 `src/config/index.ts` 中新增了三個功能開關：

```typescript
FEATURES: {
  // ... 其他功能開關
  TUTORIAL_MODE: true,     // 教學模式開關，控制是否顯示教學標籤
  USER_GUIDE: true,        // 使用說明頁面
  OPERATION_GUIDE: true    // 操作指南頁面
}
```

### 2. 後台管理新增標籤

在 AdminSystem 中新增了兩個標籤頁：
- **📖 使用說明** - 提供系統概述、快速入門、主要功能介紹、常見問題等
- **🎯 操作指南** - 按功能模組分類的詳細操作步驟說明

### 3. 主頁面整合

主頁面會根據功能開關自動顯示教學相關的功能卡片：
- 當 `USER_GUIDE` 開啟時，顯示「使用說明」卡片
- 當 `OPERATION_GUIDE` 開啟時，顯示「操作指南」卡片

## 📋 使用方法

### 管理員操作

1. **啟用/關閉教學功能**：
   - 進入「後台管理」→「功能開關」
   - 找到教學相關的開關：
     - `TUTORIAL_MODE`: 教學模式總開關
     - `USER_GUIDE`: 使用說明頁面
     - `OPERATION_GUIDE`: 操作指南頁面
   - 切換開關狀態並保存設定

2. **編輯教學內容**：
   - 教學內容在 `AdminSystem.tsx` 中的 `UserGuide` 和 `OperationGuide` 組件
   - 可以根據餐廳實際需求修改內容

### 用戶使用

1. **從主頁面訪問**：
   - 在主頁面功能區塊中點擊「使用說明」或「操作指南」
   - 自動跳轉到對應的教學頁面

2. **直接訪問**：
   - 進入「後台管理」
   - 點擊側邊欄中的「使用說明」或「操作指南」標籤

## 📖 教學內容結構

### 使用說明 (UserGuide)

- **🏠 系統概述**: 系統特色與適用場景
- **🚀 快速入門**: 三步驟入門指南
- **🎛️ 主要功能**: 各功能模組介紹
- **❓ 常見問題**: FAQ 解答
- **📞 技術支援**: 聯絡方式

### 操作指南 (OperationGuide)

分為五個功能區塊，每個區塊提供詳細的操作步驟：

1. **🍽️ 點餐操作**: 建立訂單、修改訂單、結帳流程
2. **🪑 桌位管理**: 新增桌位、狀態管理、預約管理
3. **📦 庫存操作**: 新增項目、進貨作業、庫存監控
4. **📊 報表使用**: 營收報表、商品分析、自動報表
5. **⚙️ 系統管理**: 功能開關、備份還原、資料庫編輯

## 🎨 設計特色

### 響應式設計
- 支援桌面和行動裝置
- 自動調整佈局和字體大小

### 視覺化設計

- 步驟編號和圖示增強可讀性

### 互動式導航
- 標籤式切換不同操作指南
- URL 參數支援直接跳轉到特定頁面

## 🔄 URL 參數支援

可以使用 URL 參數直接跳轉到特定教學頁面：

- `/admin?tab=userguide` - 直接跳轉到使用說明
- `/admin?tab=operationguide` - 直接跳轉到操作指南
