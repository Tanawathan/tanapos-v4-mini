# 🧪 教學系統功能測試指南

## 📋 測試目標

驗證教學系統的各項功能是否正常運作，包括功能開關控制、內容顯示、URL 跳轉等。

## 🔍 測試步驟

### 1. 基本功能測試

#### 1.1 檢查主頁面教學標籤
1. 開啟瀏覽器前往 `http://localhost:5173/`
2. 在功能區塊中應該能看到：
   - 📖 使用說明（當 USER_GUIDE 開啟時）
   - 🎯 操作指南（當 OPERATION_GUIDE 開啟時）

#### 1.2 檢查後台管理教學標籤
1. 前往 `http://localhost:5173/admin`
2. 在左側選單中應該能看到：
   - 📖 使用說明
   - 🎯 操作指南

#### 1.3 測試教學內容
1. 點擊「使用說明」標籤
2. 確認顯示完整的使用說明內容，包括：
   - 🏠 系統概述
   - 🚀 快速入門
   - 🎛️ 主要功能
   - ❓ 常見問題
   - 📞 技術支援

3. 點擊「操作指南」標籤
4. 確認顯示操作指南內容，包括：
   - 🍽️ 點餐操作
   - 🪑 桌位管理
   - 📦 庫存操作
   - 📊 報表使用
   - ⚙️ 系統管理

### 2. 功能開關測試

#### 2.1 訪問功能開關設定
1. 在後台管理中點擊「功能開關」
2. 找到教學相關的開關：
   - `TUTORIAL_MODE`: 教學模式開關，控制是否顯示教學標籤
   - `USER_GUIDE`: 使用說明頁面，提供系統基本使用指南
   - `OPERATION_GUIDE`: 操作指南頁面，提供詳細操作步驟說明

#### 2.2 測試開關控制
**測試 USER_GUIDE 開關：**
1. 關閉 `USER_GUIDE` 開關
2. 點擊「保存設定」
3. 返回主頁面 (`http://localhost:5173/`)
4. 確認「使用說明」功能卡片已隱藏
5. 返回後台管理，確認左側選單中「使用說明」標籤已隱藏

**測試 OPERATION_GUIDE 開關：**
1. 關閉 `OPERATION_GUIDE` 開關
2. 點擊「保存設定」
3. 返回主頁面
4. 確認「操作指南」功能卡片已隱藏
5. 返回後台管理，確認左側選單中「操作指南」標籤已隱藏

**恢復設定：**
1. 重新開啟 `USER_GUIDE` 和 `OPERATION_GUIDE` 開關
2. 確認功能卡片和標籤重新出現

### 3. URL 參數跳轉測試

#### 3.1 直接跳轉到使用說明
1. 在瀏覽器地址欄輸入：`http://localhost:5173/admin?tab=userguide`
2. 確認直接跳轉到使用說明頁面
3. 確認左側選單中「使用說明」標籤為選中狀態

#### 3.2 直接跳轉到操作指南
1. 在瀏覽器地址欄輸入：`http://localhost:5173/admin?tab=operationguide`
2. 確認直接跳轉到操作指南頁面
3. 確認左側選單中「操作指南」標籤為選中狀態

#### 3.3 從主頁面功能卡片跳轉
1. 在主頁面點擊「使用說明」功能卡片
2. 確認跳轉到 `/admin?tab=userguide`
3. 返回主頁面，點擊「操作指南」功能卡片
4. 確認跳轉到 `/admin?tab=operationguide`

### 4. 響應式設計測試

#### 4.1 桌面端測試
1. 在標準桌面瀏覽器中測試所有功能
2. 確認佈局正常，內容完整顯示

#### 4.2 平板端測試
1. 使用瀏覽器開發者工具模擬平板尺寸
2. 確認教學內容正常顯示
3. 測試標籤切換功能

#### 4.3 手機端測試
1. 使用瀏覽器開發者工具模擬手機尺寸
2. 確認教學內容在小螢幕上正常顯示
3. 測試觸控操作的順暢性

### 5. 互動功能測試

#### 5.1 操作指南標籤切換
1. 進入操作指南頁面
2. 測試五個分類標籤的切換：
   - 🍽️ 點餐操作
   - 🪑 桌位管理
   - 📦 庫存操作
   - 📊 報表使用
   - ⚙️ 系統管理
3. 確認內容正確切換

#### 5.2 內容互動性
1. 測試教學內容中的步驟編號顯示
2. 確認警告框和提示框正常顯示
3. 檢查內容的可讀性和格式

## 🐛 常見問題排除

### 1. 教學標籤不顯示
**可能原因：**
- 功能開關被關閉
- 瀏覽器緩存問題

**解決方法：**
1. 檢查功能開關設定
2. 清除瀏覽器緩存重新載入
3. 檢查控制台是否有錯誤訊息

### 2. URL 跳轉不正確
**可能原因：**
- URL 參數格式錯誤
- 路由配置問題

**解決方法：**
1. 確認 URL 格式：`/admin?tab=標籤名稱`
2. 檢查 React Router 配置
3. 確認 useSearchParams 正確使用

### 3. 內容顯示異常
**可能原因：**
- CSS 樣式衝突
- 組件渲染錯誤

**解決方法：**
1. 檢查瀏覽器開發者工具中的樣式
2. 確認組件沒有 JavaScript 錯誤
3. 測試不同主題下的顯示效果
