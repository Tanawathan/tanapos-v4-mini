新布局預覽
📋 左側：訂單清單
• 桌號篩選
• 訂單列表
• 滾動顯示
• 固定寬度
• 響應式設計
💰 右側：結帳詳情
• 訂單詳情
• 項目清單
• 金額計算
• 結帳按鈕
• 空狀態提示
📱 響應式設計：在手機版本中，布局會自動切換為上下堆疊模式，確保在小螢幕上也能良好顯示。

🎯 布局改進重點

  • 左右 Flex 布局替代 Grid 系統
  • 左側訂單列表固定寬度 (最大 md/lg)
  • 右側結帳詳情自適應寬度 (最小 400px)
  • 垂直滾動優化，避免內容溢出
  • 移動端自動切換為垂直堆疊
  • 更好的空狀態提示設計
  • 保持所有 UIStyle 風格兼容

📐 技術細節

CSS 類別更改：

// 主容器
<div className="flex flex-col lg:flex-row gap-6 min-h-[600px]">

// 左側容器  
<div className="flex-1 lg:max-w-md xl:max-w-lg ... flex flex-col">

// 左側列表區域
<div className="flex-1 overflow-y-auto max-h-[calc(100vh-300px)] lg:max-h-[500px]">

// 右側容器
<div className="flex-1 lg:min-w-[400px] ... flex flex-col max-h-[calc(100vh-300px)] lg:max-h-[500px]">

// 右側內容區域
<div className="p-6 flex-1 overflow-y-auto">
