⚠️ **需要執行**: 在Supabase中建立資料表  

## 🛠️ 部署步驟

### 1. 登入Supabase控制台
1. 前往: https://app.supabase.com
2. 使用現有專案憑證登入
3. 選擇您的TanaPOS專案

### 2. 執行資料庫設置
1. 在Supabase專案中，點擊左側選單的「SQL Editor」
2. 點擊「New query」建立新查詢
3. 將以下檔案的完整內容複製並貼上:
   ```
   📁 database/complete-database-setup.sql
   ```
4. 點擊「Run」執行SQL腳本
5. 確認看到「Success. No rows returned」訊息

### 3. 驗證部署
1. 回到TanaPOS系統: http://localhost:5173
2. 進入後台管理: http://localhost:5173/admin
3. 檢查「資料庫管理」頁面顯示「已連線」
4. 測試「資料庫編輯器」功能

### 4. 初始資料設定
1. 在「資料庫編輯器」中:
   - 先設定「餐廳資訊」
   - 新增「產品分類」
   - 新增「產品項目」
   - 設定「桌位」資訊
