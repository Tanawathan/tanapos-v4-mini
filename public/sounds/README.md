# 音效檔案說明

這個資料夾包含 TanaPOS V4-Mini 系統的音效檔案。

## 音效檔案列表

### 系統通知音效
- `success.mp3` - 成功操作音效
- `error.mp3` - 錯誤提示音效  
- `warning.mp3` - 警告提示音效
- `info.mp3` - 資訊提示音效

### POS 專用音效
- `new-order.mp3` - 新訂單提醒音效
- `order-ready.mp3` - 訂單完成音效

## 音效使用建議

### 音量設定
- 所有音效建議音量控制在 50% 以下
- 可在系統設定中調整音效開關

### 檔案格式
- 建議使用 MP3 格式，體積小，相容性好
- 音效時長建議 1-2 秒，避免過長

### 獲取音效
如果您需要實際的音效檔案，可以：

1. **免費資源**
   - Freesound.org
   - Zapsplat.com
   - Adobe Stock (免費素材)

2. **建議音效類型**
   - 成功：清脆的鈴聲或提示音
   - 錯誤：低沉的嗶聲
   - 新訂單：明亮的通知音
   - 訂單完成：完成提示音

3. **臨時替代**
   - 系統會自動處理音效載入失敗
   - 沒有音效檔案時不會影響功能運作

## 實作說明

音效系統已整合到通知系統中，會在以下情況播放：
- 新訂單建立
- 訂單狀態變更
- 系統警告和錯誤
- 操作成功確認
