## 📊 預期行為

### 固定套餐：
- 直接加入購物車
- 顯示成功通知

### 可選套餐：
- 開啟選擇器模態框
- 顯示 "選擇套餐內容：[套餐名稱]"
- 出現前菜、主餐、飲品、甜點等選項
- 可以選擇每個分類的產品

## 🔍 除錯資訊

開啟瀏覽器開發者工具（F12），在 Console 中應該看到：

```

開始載入套餐資料...
載入的套餐資料: [...]
有效套餐數量: X
SimplePOS: combos數量: X
```

當點擊套餐時：
```
轉換後的套餐資料: {...}
套餐類型: selectable
選擇規則數量: X
處理套餐選擇: [套餐名稱] 類型: selectable
套餐選擇規則: X
可選套餐，打開選擇器
設定 selectedCombo: {...}
設定 comboQuantity: 1
設定 showComboSelector: true
```

## ⚠️ 如果還是有問題

### 檢查清單：
1. **確認執行了資料庫修復腳本**
2. **檢查瀏覽器 Console 是否有錯誤**
3. **確認套餐類型為 'selectable'**
4. **確認套餐有選擇規則**

### 常見問題：
1. **ComboSelector 沒有顯示**
   - 檢查 showComboSelector 狀態
   - 檢查 selectedCombo 是否設置

2. **套餐沒有選擇規則**
   - 執行診斷腳本檢查
   - 手動添加選擇規則

3. **模態框被遮擋**
   - 檢查 z-index 設置
   - 確認沒有其他元素覆蓋
