# 套餐系統設置完整指南

## 快速開始 - 三步驟完成設置

### 第一步：執行資料庫腳本
1. 開啟 Supabase Dashboard
2. 前往 `SQL Editor`
3. 複製 `setup-combo-system-with-examples.sql` 的所有內容
4. 貼上並執行

### 第二步：啟動系統
```bash
npm run dev
```

### 第三步：進入管理介面
1. 開啟瀏覽器前往 `http://localhost:5173`
2. 點擊右上角的 "管理" 按鈕
3. 選擇 "套餐管理" 分頁

## 🎯 現在您有兩種設定套餐的方式

### 方式一：使用管理介面（推薦）✨
**完全視覺化操作，無需編寫 SQL**

1. **新增套餐**
   - 填寫套餐名稱、描述、價格
   - 選擇套餐類型（固定組合 / 可選組合）
   - 設定製作時間

2. **設定選擇規則**
   - 選擇分類（前菜、主餐、飲品、甜點）
   - 設定最少 / 最多選擇數量
   - 調整順序

3. **即時預覽**
   - 查看可選產品數量
   - 驗證設定正確性

### 方式二：使用 SQL（進階使用者）
參考 `setup-combo-system-with-examples.sql` 中的範例

## 📋 系統功能說明

### 套餐類型
- **固定組合**：預設固定的產品組合
- **可選組合**：顧客可以從指定分類中選擇

### 選擇規則
- **最少選擇**：顧客必須至少選擇的產品數量
- **最多選擇**：顧客最多可以選擇的產品數量
- **分類限制**：只能從指定分類中選擇

## 🔧 如果遇到問題

### 套餐沒有顯示在 POS 介面
1. 確認套餐狀態為 "可用"
2. 檢查是否有設定選擇規則
3. 確認分類中有可用的產品

### 選擇規則無效
1. 檢查分類是否存在
2. 確認該分類下有可用產品
3. 驗證最少/最多選擇數量設定

### 資料庫連接問題
1. 檢查 `.env` 檔案中的 Supabase 設定
2. 確認 RLS 政策已正確設置
3. 檢查網路連接

## 📞 技術支援

如果仍有問題，請檢查：
1. 瀏覽器開發者工具的 Console 錯誤
2. Supabase Dashboard 的 Logs
3. 確認所有必要的表格和資料都已創建
