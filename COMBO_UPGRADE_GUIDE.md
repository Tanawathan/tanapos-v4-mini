## 🛠️ 使用步驟

### 第一步：設定可選擇套餐

1. **進入套餐管理**
   - 開啟 http://localhost:5173
   - 點擊「管理系統」→「🍽️ 套餐管理」

2. **創建可選擇套餐**
   - 點擊「新增套餐」
   - 選擇「可選擇套餐」類型
   - 填入名稱：如「夏日套餐」
   - 設定基礎價格：如 NT$280

3. **設定選擇規則**（目前需在 Supabase 中手動設定）
   - 在 `combo_selection_rules` 表格中新增規則
   - 例如：選擇前菜（category_id=前菜分類ID, min=1, max=1）

### 第二步：測試套餐選擇

1. **整合到點餐介面**（待開發）
   - 在 SimplePOS 中顯示套餐
   - 點擊套餐時開啟 `ComboSelector`
   - 客戶選擇套餐內容

2. **套餐選擇流程**
   - 顯示所有選擇規則（前菜、主餐、飲品、甜點）
   - 從每個分類中選擇產品
   - 計算總價（基礎價格+額外費用）
   - 確認選擇並加入購物車

## 📊 資料庫結構說明

### 套餐產品表 (combo_products)
```sql
- combo_type: 'fixed' | 'customizable'
- 固定套餐：傳統固定內容套餐
- 可選擇套餐：客戶自選內容套餐
```

### 選擇規則表 (combo_selection_rules)
```sql
- combo_id: 套餐ID
- category_id: 分類ID（如前菜、主餐）
- selection_name: 顯示名稱（如"選擇前菜"）
- min_selections: 最少選擇數量
- max_selections: 最多選擇數量
- is_required: 是否必選
```

### 選擇選項表 (combo_selection_options)
```sql
- rule_id: 規則ID
- product_id: 產品ID
- additional_price: 額外價格
- is_default: 是否預設選項
```

## 🎯 使用範例

### 夏日特色套餐設定
```
基礎價格: NT$280
選擇規則:
1. 選擇前菜 (1-1項，必選)
   - 凱薩沙拉 (+NT$0)
   - 蒜香麵包 (+NT$0)
   - 酥炸雞翅 (+NT$20)

2. 選擇主餐 (1-1項，必選)
   - 檸檬雞排 (+NT$0)
   - 蜜汁豬排 (+NT$10)
   - 鮭魚排 (+NT$30)

3. 選擇飲品 (1-1項，必選)
   - 可樂 (+NT$0)
   - 檸檬汁 (+NT$10)
   - 現打果汁 (+NT$25)

4. 選擇甜點 (0-1項，可選)
   - 提拉米蘇 (+NT$35)
   - 巧克力蛋糕 (+NT$30)
```

### 客戶選擇範例
```
夏日特色套餐 基礎價格: NT$280
+ 蒜香麵包 (+NT$0)
+ 蜜汁豬排 (+NT$10)
+ 現打果汁 (+NT$25)
+ 提拉米蘇 (+NT$35)
------------------------
總計: NT$350
```
